TAREA DE LENGUAJES PARA DES Y SOP SIST

¿Qué es un protocolo Http?
Es cuando un cliente solicita al servidor una conexión para mantener la comunicación y realizar de manera segura y efectiva el intercambio de datos.

¿Qué es un Query String?
Es la parte del URL de una página, sirve para facilitar la experiencia del usuario al momento de realizar una búsqueda por medio de una URL.

¿Qué es un Verbos HTTP?
Es un conjunto de elementos, cada uno realiza una acción diferente, con el cuál el servidor indentifica que función debe cumplir.

¿Qué es un DNS?
Es un protocolo que tiene como función conocer donde se encuentra alojado el contenido de una web, en la cual se traduce el dominio a una IP, y el motivo por el cuál no se usa directamente la IP es debido a que es dificil de recordar.

¿Qué es un CDN?
Es un grupo de servidores que se encuentran alrededor del mundo que ayudan a agilizar el proceso de entrega del contenido de un servidor, debido a que la distancia entre un usuario y el servidor donde se aloja la web provoca una demora al momento de cargar el contenido del mismo.
